* HexQL

HexQL is a macOS app containing a Quick Look preview extension.
It seems that its not possible anymore to implement a generic quicklook extension
for public.data or public.content to catch all UTI's without more specific quicklook plugins.

* Installation

#+begin_src sh
rake build install
#+end_src

This will copy the =HexQL.app= to =~/Applications=.

* Testing
Use =qlmanage -p file.hexql=
